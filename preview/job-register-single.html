⚖️ PC vs Mobile — 어떤 차이가 있나

💻 PC (3-column)
  • 한 화면에 모든 정보 — 스크롤 최소
  • 매장 선택 + 진행중 업무 + LINE 미리보기 동시 비교
  • 카테고리 12개를 한 줄에 노출 (flex-wrap)
  • 키보드 단축키 (Tab 이동, Ctrl+Enter 등록) 최적
📱 Mobile (수직 섹션)
  • 섹션별 카드로 명확한 구분
  • 카테고리 3×4 그리드로 모두 노출 — 한 번에 탭
  • 거래처 담당 박스 색상 강조
  • 하단 [등록] 버튼 항상 고정
  • 입력 필드 크게 (터치 영역 42px+)
  • 한 손 입력 가능 — 자유 순서
💡 결론 — 동일한 데이터 모델·동일한 핸들러를 공유. 뷰포트 너비로 자동 분기하거나 (@media), URL 로 명시적 토글 (?ui=pc vs ?ui=mobile) 둘 다 가능. 코드 중복 없이 두 레이아웃 유지 가능하도록 동일 컴포넌트 + 다른 CSS Grid 적용 권장.
