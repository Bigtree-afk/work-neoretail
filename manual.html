① 소개

네오리테일 작업관리 시스템은 가맹점 신규 오픈, 상담, AS, VAN 서류 접수까지 한 번에 관리하는 사내 업무 도구입니다.

주요 기능

  • 신규/상담 통합 관리 — 상담 → 납품 결정 → 신규 진행 → 종료까지 일관된 흐름
  • 월간 캘린더 + 작업 카드 — 일정과 진행 현황을 한 화면에서
  • 장비 카탈로그 — 옵션 단계까지 모든 직원이 공유
  • VAN 서류 공개 접수 — 가맹점/제출자가 로그인 없이 모바일로 사진/PDF 업로드
  • 클라우드 동기화 — 모든 직원 PC·모바일에서 데이터 자동 공유
💡 모든 데이터는 Cloudflare 클라우드 KV 에 자동 저장되어 PC를 옮겨도 그대로 보입니다.

② 로그인 / 권한

화이트리스트에 등록된 구글 계정으로만 접속이 가능합니다.

로그인 방법

  1. 우측 상단 로그인 버튼 클릭
  2. 구글 계정 선택
  3. 관리자가 등록한 화이트리스트 이메일이면 자동 입장 / 아니면 차단

권한 종류

  • 일반 직원 — 신규/상담/점포/작업/장비/VAN서류 모든 업무 수행 가능
  • 관리자 — 위 + 직원 추가/삭제, API 키 관리, 데이터 초기화
접속이 차단되면 관리자에게 본인 구글 이메일을 알려 화이트리스트 등록을 요청하세요.

④ 신규/상담

상담 → 납품 결정 → 신규 진행 → 종료의 전체 영업 사이클을 관리합니다.

주요 액션

  • + 신규 등록 — 즉시 진행할 신규 가맹점 등록
  • + 상담 등록 — 결정 전 상담 단계 매장 등록 (납품 결정 시 자동으로 신규로 전환)
  • 💬 상담 목록 — 진행 중인 상담 조회
  • ₩ 견적 작성 — 견적서 화면으로 이동

필터 탭

  • 진행 전체 (기본) — 미완료 작업만 표시
  • 예정 / 진행중 — 세부 상태별
  • 📦 지난 완료 — 종료 처리된 작업 보관소

매장 상세 모달

점포명/행 어디든 클릭 → 모달이 열림. 모달 안에서:

  • 📅 주요 일정 (설치 / 가오픈 / 오픈일) — 입력 즉시 자동 저장
  • 📍 주소 직접 수정 가능
  • 👤 매장 담당자 다중 등록 + ⭐대표 연락망 지정
  • 📦 설치 장비 추가/수량 변경/판매가 수정 / 설치 완료 체크
  • 📝 비고 인라인 수정 + 메모 추가 (KST 타임스탬프 자동 기록)

📅 미니 캘린더 (이번주 + 다음주)

화면 상단에 2주 일정이 보이고, 작업을 클릭하면 매장 상세로 바로 이동합니다.

💡 상담은 영업 단계 (구속력 낮음), 신규는 실제 작업 단계입니다. 동일 매장이 상담 → 신규로 자연스럽게 이어집니다.

⑤ 점포 관리

거래 중인 모든 가맹점 마스터 데이터.

기능

  • 매장 등록 — 점포명/사업자/대표/연락처/주소/VAN/POS·KIO 등 정보
  • 검색 — 상단 검색창 (음성 인식 🎙 지원)
  • 매장 클릭 — 상세 정보 팝업 (설치 장비, 작업 이력)
  • 모바일 가로 스크롤 — 좌측 점포명은 고정, 주소/상태 등은 스크롤로 확인
💡 매장 목록은 최근 등록순으로 정렬되어 새로 추가한 점포가 가장 위에 보입니다.

⑥ 작업/일정

월간 캘린더와 작업 카드를 함께 관리합니다.

월간 캘린더 (상단)

  • ◀ 전월 / 다음월 ▶ / 오늘 버튼
  • 색상 범례: 신규/POS교체/VAN/AS + 🔧 설치/🌅 가오픈/🎉 오픈
  • 날짜에 작업 점이 표시됨 — 일정이 많은 날은 +N 표시

작업 카드 (하단)

  • 각 카드: 점포명 / 유형 / 작업일 / 담당자 / 상태
  • 카드 클릭 → 매장 상세 모달
  • 카드 우측 하단 ✓ 종료 버튼 → 즉시 종료 처리
  • 필터 칩: 전체 / 신규 / POS교체 / VAN교체 / AS / 완료
  • 💬 Line 메시지 가져오기 — 라인 메시지에서 작업 일괄 등록

⑦ 장비 재고

장비 카테고리별 보유 / 부족 현황을 파악합니다. 카탈로그 자체 관리는 마이페이지에서.

  • 좌측: 카테고리 그룹
  • 각 항목: 장비명 / 모델 / 재고 / 부족 표시
  • 상단 검색창으로 장비명 빠른 찾기
  • 재고 부족만 보기 토글 버튼

⑧ VAN서류 (공개 접수처)

로그인 없이 누구나 모바일/PC로 가맹/변경 서류를 사진·PDF로 업로드합니다.

접수 카테고리

  • 카드가맹 신규 → 개인사업자/법인사업자 → 1인대표/공동대표
  • 정보 변경 → 대표자/상호/주소/계좌 변경
  • 해지
  • 기타 서류

사용자(제출자) 흐름

  1. 매장명·제출자 입력
  2. 업무 종류 단계 선택 (예: 카드가맹 신규 → 개인사업자 → 1인대표)
  3. 준비할 서류 목록 확인 후
  4. 📷 카메라 / 🖼 갤러리 / 📎 PDF 한꺼번에 선택해 추가
  5. [전체 업로드] 클릭

한도 / 보관

  • 파일당 최대 24MB / 제출당 합계 50MB
  • 이미지는 자동 압축 (긴 변 2400px / JPEG 92%)
  • 담당자 [확인] 후 7일 경과 시 자동 삭제
  • 미확인 상태로는 절대 삭제되지 않음

직원의 서류 검토

  • 대시보드 좌측 📑 신규 VAN 서류 카드 — NEW 표시
  • 마이페이지의 보라색 섹션 — 모든 서류 목록
  • 항목 클릭 → 상세 모달 → 썸네일/체크박스/[선택 다운로드][전체 다운로드]
  • 다운로드 시 부모 폴더 선택 → 매장명_서류종류_날짜/ 하위폴더 자동 생성 (Chrome/Edge)
  • [확인 처리] → 알림에서 사라짐 + 7일 후 자동 삭제 카운트다운 시작

⑨ 마이페이지

우측 상단 프로필 → 📦 마이페이지

섹션

  • 📑 VAN 서류 확인 — 미확인 건수 뱃지 + 항목 클릭 → 다운로드/확인 처리
  • ☁ 다중 PC 동기화 — 다른 PC에서 작업/점포가 안 보일 때 [작업 동기화 / 받기], [점포 동기화 / 받기]
  • 📦 장비 카탈로그 관리 — 신규 등록 시 노출되는 장비 항목 추가/수정/삭제 (모든 직원 가능)

장비 카탈로그 옵션

  • 각 장비에 옵션 단계 텍스트 형식 입력 가능
  • 예: 색상: 검정, 흰색, 회색 ↵ 크기: 소, 중, 대
  • 변경 즉시 자동 클라우드 동기화 → 모든 직원 PC에 반영

⑩ 관리자 페이지

관리자 권한이 있는 사용자만 사용. 우측 상단 ⚙️ 톱니바퀴 또는 프로필 메뉴.

주요 항목

  • 등록 직원 관리 — 직원 추가 / 이름·직책·전화·역할 인라인 편집 / 삭제
  • API 키 관리 — Claude API 키 (대시보드 AI 분석 용)
  • 접근 허용 이메일 (화이트리스트) — 직원 추가 시 자동 등록됨
  • Google OAuth — Google Client ID 설정
  • AI 대시보드 분석 — Claude 기반 신규/상담 트렌드 요약
  • ☁ 클라우드 동기화 — 작업/점포/카탈로그/화이트리스트 풀&푸시
  • 💾 자동 백업 — 브라우저 로컬 자동 저장 + 수동 export/import
  • ⚠ 데이터 초기화 — 위험: 모든 데이터 영구 삭제 (확인 다이얼로그)
⚠ 데이터 초기화는 되돌릴 수 없습니다. 관리자도 신중히 사용하세요.

⑪ 업무 흐름 한눈에

가맹점 영업 → 오픈까지

  1. 잠재 고객 발굴 → 상담 등록 (신규/상담 메뉴)
  2. 장비/일정 협의 → 상담 메모/일정 갱신
  3. 납품 결정 → → 신규 전환 클릭 → 신규 작업으로 변환
  4. 설치 일정 입력 (설치 / 가오픈 / 오픈일)
  5. 장비 추가/수량 입력 → 설치 완료 체크
  6. 오픈 완료 후 ✓ 종료 → 지난 완료로 이동

VAN 서류 처리

  1. 가맹점/제출자 → /vandocs.html 공개 페이지에서 사진·PDF 업로드
  2. 대시보드/마이페이지 알림 확인
  3. 썸네일/다운로드 검토 → [확인] 처리
  4. 7일 경과 시 자동 정리

AS 처리

  1. 대시보드 AS 미처리 카드/표에서 신규 접수 확인
  2. 행/카드 클릭 → 매장 상세에서 메모, 일정 추가
  3. 완료 후 ✓ 종료

⑫ 자주 묻는 질문

Q. 다른 PC에서 입력한 작업이 안 보입니다.

마이페이지 상단 ☁ 작업 받기 / 점포 받기 클릭 후 새로고침. 보통 자동 동기화되지만 캐시가 남아있을 때가 있습니다.

Q. 새로 추가한 장비 카탈로그가 다른 직원 PC에서 안 보여요.

마이페이지에서 ☁ 카탈로그 동기화 클릭. 변경은 자동 push 되지만 다른 직원이 페이지를 다시 열어야 받아옵니다.

Q. VAN 서류 다운로드 시 매번 폴더를 선택해야 하나요?

처음 한 번 부모 폴더(예: 다운로드)를 선택하면, 같은 세션 내에서는 자동 저장됩니다. 새 세션에서는 다시 선택이 필요합니다 (Chrome 보안 정책).

Q. 잘못 종료 처리한 작업을 되살리려면?

신규/상담 → 📦 지난 완료 탭 → 해당 작업 행 우측 ↩ 되돌리기 버튼.

Q. 마이크 권한이 거부되어 음성 검색이 안 됩니다.

주소창 좌측 자물쇠 🔒 → 사이트 설정 → 마이크 허용. Chrome/Edge 권장.

Q. 모바일에서 메뉴가 안 보입니다.

좌측 상단 로고를 누르면 메뉴가 펼쳐집니다. 우측 햄버거 ☰ 버튼도 동일 동작.

Q. 데이터가 사라졌어요!

관리자에게 문의. 마이페이지 💾 자동 백업의 export 파일이 있다면 import 로 복구 가능. 클라우드 동기화 시점에 따라 다른 PC에서 받아올 수도 있습니다.
